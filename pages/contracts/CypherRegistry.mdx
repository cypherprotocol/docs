# CypherRegistry

Registry that stores mappings between protocol contracts and their assigned escrow

## Functions

### createEscrow

```solidity
function createEscrow(
  address protocol,
  address token,
  uint256 tokenThreshold,
  uint256 timeLimit,
  address[] memory oracles
) public architectOnly(protocol) returns (address)
```

Creates a new escrow contract and registers it in the registry

### attachEscrow

```solidity
function attachEscrow(address escrow, address protocol) public architectOnly(protocol)
```

Attaches an existing escrow contract to a protocol
