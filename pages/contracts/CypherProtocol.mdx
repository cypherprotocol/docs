# CypherProtocol

Primary contract to inherit from in order to interact with Cypher

## Constructor

```solidity
constructor(
  string memory _protocolName,
  address _architect,
  address _registry
)
```

### Parameters

- `_protocolName` - The name of the protocol
- `_architect` - The address of the architect, which is the user who will be able to create and assign an escrow
- `_registry` - The address of the CypherRegistry found [here](/contracts/deployment-addresses)
