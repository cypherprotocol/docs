# What does Cypher protect?

Cypher protects virtually all contracts that store value: assigned, approved, or storing of actual network tokens in a smart contract.

- `onERC1155Received()`
- `onERC721Received()`
- `fallback()`
- `receive()`

## Example contracts

- Liquidity pools
- NFT pools
- NFT minting contract
- Escrow contracts
- Debt protocol contracts
