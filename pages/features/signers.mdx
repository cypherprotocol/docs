# Signers

Cypher supports many signing types, allowing you to integrate with virtually any auth system your protocol wants to use. These signers allow and deny specific transactions, depending on the rules that you set in your Cypher Escrow contracts.

- Multi-sig wallets
- Centralized exchange wallets
- Governance contracts
